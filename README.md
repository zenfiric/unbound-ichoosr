# unbound-pom-poc

A proof-of-concept AI agent implementation using Microsoft AutoGen framework.

## Overview

This project demonstrates the use of AutoGen to create AI agents that can collaborate and solve tasks together.

## Setup

1. Create a virtual environment:

   ```python
   pyenv activate ichoosr
   ```

2. Install the required packages:

   ```python
   pip install -r requirements.txt
   ```
